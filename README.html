使い方

                                        
                                      コマンド

                                       /set spawns [name]   :   各島のスポーン地点を設定します。
                                       各島のスポーンさせたいところに立ち /set spawns 好きな名前    をうちます。
                                     
                                       /set wait  :  ゲーム開始までの待機場所を設定します。
                                       待機させたい場所で /set wait  をうちます。

                                       /dispawn   :  プレイヤーがゲームオーバーになった際の観戦開始場所を設定します。
                                       プレイヤーはスぺクテイターモードで終了まで観戦します。

                                       /opt [name]  : チェストの設定をします。
                                       /opt 好きな名前  を打ち　チェストにタッチします。
                                     



                                    Configの設定

                                        SaveConfigMessageをtrueにするとConfigにデータを保存した際にCONSOLEに表示されます。
               
                                        待ち時間はゲーム開始までの時間を秒数で指定します。
      
                                        最大人数はゲームに参加できる最大人数です。
                                        最低人数はゲームを開始できる人数です。
  
                                    その他

                                        受付は骨をもってタッチで設定できます。
                                        会場はDEFAULTのワールド以外にしてください。
                                        つまりロビーと会場は別々のワールドにしてください。
                                        また、プレイヤーの座標が Y = -1 になるとスポーン地点に戻ります。
                                        AlwaysSpawnを入れることを推奨します。
